# Project Overview

CloudWiki is a Nextcloud app focused on raw Markdown editing with Obsidian-style wikilinks and backlinks.

## Source Docs
- Scope document: `../scope.md`
- TODO backlog: `../TODO.md`
- Repository readme: `../README.md`

## Core Principles
- Keep notes as plain `.md` files in Nextcloud storage.
- Respect Nextcloud auth, ACLs, sharing, and version history.
- Preserve interoperability with Obsidian for the same synced folder.
