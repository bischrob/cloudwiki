# MVP Scope

## In Scope
- Raw Markdown editor (single pane, no live preview).
- `[[wikilinks]]` autocomplete and navigation.
- Create note from unresolved wikilink.
- Backlinks panel from indexed note links.
- Nextcloud-native file handling with permission enforcement.

## Out of Scope
- Markdown preview rendering.
- Graph view.
- Plugin marketplace.

## Compatibility Requirement
A user can edit notes in Obsidian (pointed at a Nextcloud-synced folder) and edit the same files in CloudWiki without conversion or lock-in.
